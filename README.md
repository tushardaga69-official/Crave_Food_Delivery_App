# 🍔 Crave – Food Delivery Mobile App UI

A modern and user-friendly Food Delivery Mobile App UI designed in Figma with a clean and minimal interface focused on smooth user experience and intuitive navigation.

---

## 📱 Screens Included

- ✨ Splash / Onboarding Screen
- 🔐 Login / Signup Screen
- 🍕 Home Screen
- 🍔 Food Details Screen
- 🛒 Cart Screen

---

## 🎯 Project Objective

The goal of this project was to design a visually appealing and user-centric food delivery application that provides a seamless ordering experience for users.

---

## 💡 Problem Statement

Users often face cluttered interfaces and complicated ordering flows in food delivery applications. The challenge was to create a simple, modern, and intuitive UI that improves usability and enhances the overall user experience.

---

## 🚀 Solution

Designed a clean and modern mobile app interface with:
- Easy navigation
- Consistent design system
- Minimal layout
- Clear visual hierarchy
- Smooth user journey from browsing to checkout

---

## 🎨 Design Process

1. Research & Inspiration
2. User Flow Planning
3. Wireframing
4. UI Design
5. Prototyping

---

## 🛠️ Tools Used

- Figma
- Auto Layout
- Components & Variants

---

## 🎨 Design Style

- Minimal UI
- Soft shadows
- Rounded corners
- Orange accent palette
- Modern typography

---

## 📌 Features

- Clean onboarding experience
- Simple login/signup flow
- Interactive food browsing
- Detailed food preview
- Easy cart management
